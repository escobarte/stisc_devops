| Инструмент | Назначение                 | Примечания                                |
| ---------- | -------------------------- | ----------------------------------------- |
| `htop`     | Процессы, CPU, RAM         | Удобный интерфейс                         |
| `dstat`    | CPU, RAM, I/O, сеть        | Заменяет vmstat, iostat и ifstat          |
| `glances`  | Все ресурсы в одном окне   | Требует установки (`pip install glances`) |
| `netdata`  | Графики в реальном времени | Веб-интерфейс                             |
| `nmon`     | Мониторинг и запись в файл | Удобен для отчётов                        |


NFS – общий каталог для 3 VM
Настроить NFS-сервер и подключить к нему 2 клиента.
Теория:
* `nfs-kernel-server`, `/etc/exports`, `mount -t nfs`, `showmount`

FTP-сервер
Настроить простой FTP для загрузки/скачивания файлов.
Теория:
* `vsftpd`, `ftp`, `anonymous access`, конфигурация `/etc/vsftpd.conf`
