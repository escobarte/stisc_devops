5. Simple GitLab CI Pipeline
In GitLab, the CI/CD pipeline is defined using a .gitlab-ci.yml
stages: Defines the order of pipeline steps (test in this case).
test-job: The name of the job.
script: The list of shell commands to run in that job.
'''
stages:
 -
  test
test-job:
 stage: test
 script:
 - echo "Running tests..."
 - echo "CI is workgin, grea job [Serge]"
